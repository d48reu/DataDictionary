About the MDC Open Data Encyclopedia

Mission

The MDC Open Data Encyclopedia makes Miami-Dade County's open data catalog discoverable and understandable by non-technical residents. Every dataset gets an AI-generated plain-English description, quality score, and change tracking so that residents, journalists, researchers, and community organizations can find and evaluate public data without needing technical expertise.

How It Works

  • Data Collection: Metadata is pulled from Miami-Dade County's ArcGIS Hub open data portal, capturing dataset titles, descriptions, schemas, and update timestamps.
  • AI Enrichment: Claude AI (Anthropic) generates plain-English descriptions, potential use cases, keywords, department attribution, and civic relevance scores for each dataset.
  • Quality Audit: Automated scoring evaluates each dataset across three dimensions: freshness (how recently updated), completeness (whether metadata is filled in), and documentation quality.
  • Change Detection: Automated comparison between data pulls tracks new, removed, and modified datasets so you can see how the catalog evolves over time.
  • Static Site: This website is generated from the enriched database and updated weekly to reflect the latest state of the open data catalog.

Disclaimer

This is NOT an official Miami-Dade County product.

This site is an independent civic technology project. It aggregates and presents publicly available metadata from Miami-Dade County's open data portal.

AI-generated descriptions are produced by Claude (Anthropic) and may contain inaccuracies. Always verify critical information with the original data source.

Quality scores are automated assessments and do not reflect official data quality evaluations.
