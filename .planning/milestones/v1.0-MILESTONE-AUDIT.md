### Integration Bug (1)

**BUG-01: Grade badge CSS class mismatch on browse pages**
- `context.py` sets `ds["grade_css"]` per dataset
- `browse.html` reads `{{ ds.grade_class|default('grade-none') }}`
- Result: Grade badges render with `grade-none` class (gray/unstyled) on browse pages
- Dataset detail pages and related datasets correctly use `grade_css`/`grade_class` respectively
- **Severity:** Cosmetic — no crash, no data loss, grade info visible on detail pages
- **Fix:** One-line change in `browse.html` (grade_class -> grade_css)
