**DimensionResult dataclass:**
```python
@dataclass
class DimensionResult:
    score: float          # 0.0-1.0
    findings: list[str]   # Human-readable findings
    scored: bool          # False if N/A (dimension couldn't be scored)
```

**FREQUENCY_DAYS dict:** Maps lowercase frequency strings to expected day intervals. Include: daily(2), weekly(14), biweekly(21), monthly(45), quarterly(120), biannually(210), annually(400), as needed(None), static(-1), one-time(-1). Auto-exempt sentinel is -1, None means use fallback tiers.

**normalize_frequency(freq_str) function:** Case-insensitive exact match first, then substring fallback (if "daily" in string -> daily, if "week" in string -> weekly, etc.). Returns int, -1, or None.

**score_freshness(updated_at, update_freq, now) function:**
- Accept `now: datetime` parameter for testability (never hardcode datetime.now)
- No updated_at -> score 0.0, finding "No last-updated date available"
- Parse updated_at via datetime.fromisoformat(), handle timezone-naive by assuming UTC
- Check normalize_frequency(update_freq):
  - -1 (static/one-time) -> score 1.0, scored=False, finding "Static/one-time dataset (exempt)"
  - Positive int -> ratio = days_since / expected_days. ratio<=1.0: 1.0; ratio<=2.0: 0.7 + finding; ratio<=3.0: 0.4 + finding; else: 0.1 + finding
  - None -> fixed tiers: <=90d: 1.0; <=180d: 0.6 + finding; <=365d: 0.3 + finding; else: 0.1 + finding

**score_completeness(dataset) function:**
- Check 5 metadata fields, each scored 0-1 with partial credit:
  - description: empty=0, <50 chars=0.3 + finding, <100 chars=0.7 + finding, else=1.0
  - row_count: None=0.3 + "Row count not available" finding, 0=0.0 + "Empty dataset (0 rows)", else=1.0
  - tags: present and not "[]"=1.0, else=0 + finding
  - category: present=1.0, else=0 + finding
  - publisher: present=1.0, else=0 + finding
- Score = sum of partial credits / 5

**score_documentation(column_count, documented_count) function:**
- column_count 0 -> score 0.0, scored=False, finding "No column metadata available"
- ratio = documented_count / column_count
- <0.25 -> finding "Poor column documentation"
- <0.75 -> finding "Partial column documentation"
- Score = ratio

**compute_composite(freshness, completeness, documentation) function:**
- Base weights: freshness=0.50, completeness=0.30, documentation=0.20
- Only include dimensions where scored=True in the weighted average
- Redistribute weight proportionally among scored dimensions
- If no dimensions scored -> return (0.0, "F") with finding "Insufficient metadata to score"
- Return (composite_score, letter_grade)

**GRADE_THRESHOLDS:** [(0.90, "A"), (0.75, "B"), (0.60, "C"), (0.40, "D"), (0.0, "F")]

**score_to_grade(score) function:** Iterate thresholds, return first match.

**audit_dataset(dataset_row, now) function:**
- Convenience function that takes a dict from get_all_datasets_for_audit and datetime
- Calls score_freshness, score_completeness, score_documentation
- Calls compute_composite
- Returns dict with: composite_score, letter_grade, staleness (freshness.score), completeness (completeness.score), documentation (documentation.score), findings (list of all finding strings from all dimensions)
