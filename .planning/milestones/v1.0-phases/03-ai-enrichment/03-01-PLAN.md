**`src/mdc_encyclopedia/enrichment/client.py`**: Create the Anthropic client wrapper with three functions:

1. `create_enrichment_client(model: str = DEFAULT_MODEL, max_retries: int = 5) -> anthropic.Anthropic`:
   - Import and call `load_dotenv()` from python-dotenv to auto-load `.env` file
   - Check `os.environ.get("ANTHROPIC_API_KEY")` -- if missing, raise `ValueError` with clear message: "ANTHROPIC_API_KEY not set. Set it in your environment or add it to a .env file."
   - Return `anthropic.Anthropic(api_key=api_key, max_retries=max_retries)`

2. `enrich_dataset(client: anthropic.Anthropic, model: str, dataset: dict, columns: list[dict]) -> EnrichmentResult`:
   - Import `SYSTEM_PROMPT`, `build_dataset_prompt` from prompts module
   - Import `EnrichmentResult` from models module
   - Build the user prompt via `build_dataset_prompt(dataset, columns)`
   - Call `client.messages.parse(model=model, max_tokens=1024, system=SYSTEM_PROMPT, messages=[{"role": "user", "content": prompt}], output_format=EnrichmentResult)`
   - Return `result.parsed_output`

3. `estimate_cost(client: anthropic.Anthropic, model: str, datasets: list[dict], columns_by_dataset: dict[str, list[dict]]) -> dict`:
   - For each dataset, build the prompt via `build_dataset_prompt(dataset, columns_by_dataset.get(dataset["id"], []))`
   - Call `client.messages.count_tokens(model=model, system=SYSTEM_PROMPT, messages=[{"role": "user", "content": prompt}])` for each
   - Sum input_tokens across all datasets
   - Estimate total output tokens as `ESTIMATED_OUTPUT_TOKENS * len(datasets)`
   - Look up pricing from `MODEL_PRICING` dict (fall back to DEFAULT_MODEL pricing if model not found)
   - Return dict with keys: `input_tokens`, `output_tokens_est`, `input_cost`, `output_cost_est`, `total_est`, `dataset_count`
   - Add a 0.1s sleep between count_tokens calls to avoid rate limiting the token counting endpoint

**`src/mdc_encyclopedia/db.py`**: Add three new functions (DO NOT modify existing functions):
