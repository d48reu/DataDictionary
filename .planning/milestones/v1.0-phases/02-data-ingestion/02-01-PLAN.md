Functions:

1. `create_client() -> httpx.Client` -- Create an httpx.Client with a 30-second timeout and standard headers (User-Agent: "mdc-encyclopedia/0.1.0"). Return the client (caller manages lifecycle).

2. `fetch_catalog_page(client: httpx.Client, start_index: int = 1) -> dict` -- Decorated with `@retry(stop=stop_after_attempt(3), wait=wait_exponential(multiplier=2, min=2, max=8), reraise=True)`. Calls `time.sleep(RATE_LIMIT_SECONDS)` before making the request. GETs the search endpoint with params `{"limit": PAGE_SIZE, "startindex": start_index}`. Calls `response.raise_for_status()`. Returns the parsed JSON response.

3. `fetch_all_datasets(client: httpx.Client) -> Iterator[tuple[dict, int, int]]` -- Generator that paginates through the catalog. First call gets `numberMatched` (total count). Yields `(feature, current_index, total_count)` tuples for each feature in each page. Follows `rel=next` links from the response `links` array to get the next `startindex`. Stops when no `rel=next` link is found or features list is empty.

4. `detect_duplicate_titles(datasets: list[dict]) -> list[tuple[str, list[str]]]` -- Takes a list of normalized dataset dicts. Groups by normalized title (`title.strip().lower()`). Returns a list of `(normalized_title, [dataset_ids])` for any title that appears more than once. Ignores None/empty titles. Uses exact match only on the normalized form (per locked decision: normalized title matching, no fuzzy). This satisfies within-catalog dedup detection (INGEST-06 reinterpreted for single portal -- research confirmed no Socrata portal exists, so cross-portal dedup is moot; within-catalog title matching detects datasets that may appear under the same name).

Use synchronous httpx (NOT async). Per research: at 1 req/sec, async adds complexity with zero benefit.

**src/mdc_encyclopedia/ingestion/normalizer.py:** Create the data normalization module.

Constants:
- `ESRI_TYPE_MAP` dict mapping ArcGIS field types to simplified types: `esriFieldTypeOID` -> `"integer"`, `esriFieldTypeInteger` -> `"integer"`, `esriFieldTypeSmallInteger` -> `"integer"`, `esriFieldTypeDouble` -> `"number"`, `esriFieldTypeSingle` -> `"number"`, `esriFieldTypeString` -> `"text"`, `esriFieldTypeDate` -> `"date"`, `esriFieldTypeGeometry` -> `"geometry"`, `esriFieldTypeGlobalID` -> `"text"`, `esriFieldTypeGUID` -> `"text"`, `esriFieldTypeBlob` -> `"binary"`, `esriFieldTypeXML` -> `"text"`.

Functions:

1. `strip_html(text: str) -> str` -- Strip HTML tags using `re.sub(r'<[^>]+>', '', text)`, then decode HTML entities with `html.unescape()`, then strip leading/trailing whitespace. Return empty string if input is None or empty.

2. `ms_to_iso(ms_timestamp: int | None) -> str | None` -- Convert millisecond Unix timestamp to ISO 8601 string using `datetime.fromtimestamp(ms / 1000, tz=timezone.utc).isoformat()`. Return None if input is None.

3. `normalize_hub_dataset(feature: dict) -> dict` -- Convert ArcGIS Hub Search API feature to a dict matching the datasets table columns. Field mapping (from research):
   - `id`: `feature["id"]`
   - `source_portal`: `"arcgis_hub"` (hardcoded -- single portal reality)
   - `source_url`: `f"https://opendata.miamidade.gov/datasets/{feature['id']}"`
   - `title`: `props["title"]`
   - `description`: `strip_html(props.get("description", ""))`
   - `category`: Parse from `props["categories"]` list -- find first entry matching `/Categories/...`, use the last segment. None if no categories.
   - `publisher`: `props.get("source", props.get("owner", ""))`
   - `format`: `props.get("type")` (e.g., "Feature Service", "File Geodatabase")
   - `created_at`: `ms_to_iso(props.get("created"))`
   - `updated_at`: `ms_to_iso(props.get("modified"))`
   - `row_count`: `None` (not available from search API)
   - `tags`: `json.dumps(props.get("tags", []))`
   - `license`: `strip_html(props.get("licenseInfo", ""))`
   - `api_endpoint`: `props.get("url")` (service URL; None for File Geodatabases)
   - `bbox`: `json.dumps(feature.get("geometry"))` if geometry exists, else None
   - `download_url`: Construct `f"https://opendata.miamidade.gov/api/download/v1/items/{feature['id']}/csv?layers=0"` for Feature Services, None for File Geodatabases
   - `metadata_json`: `json.dumps(props)`

4. `normalize_field(field: dict, dataset_id: str, layer_name: str = "") -> dict` -- Convert ArcGIS REST field to a dict matching the columns table. Map field type via `ESRI_TYPE_MAP` (default to raw type string if not found). Include `dataset_id`, `name` (field name), `data_type`, and `description` (use alias, prepend layer name if provided: `f"{layer_name}: {alias}"`).

**src/mdc_encyclopedia/db.py:** Add two new functions to the existing module. DO NOT modify existing functions or schema.

1. `upsert_dataset(conn: sqlite3.Connection, dataset: dict) -> str` -- Check if dataset exists by id. Execute `INSERT OR REPLACE INTO datasets` with all columns from the normalized dict. Set `pulled_at` to `datetime('now')`. Return `"new"` or `"updated"`.

2. `upsert_columns(conn: sqlite3.Connection, dataset_id: str, columns: list[dict]) -> int` -- Delete existing columns for the dataset_id first (`DELETE FROM columns WHERE dataset_id = ?`). Then insert each column. Return the count of columns inserted. Uses `INSERT INTO columns (dataset_id, name, data_type, description) VALUES (?, ?, ?, ?)`.
