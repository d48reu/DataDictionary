**In src/mdc_encyclopedia/cli.py:**

1. Add a `get_default_db_path()` function near the top (after imports, before the `cli` group) that returns an absolute path:
   ```python
   def get_default_db_path() -> str:
       """Return absolute default database path using XDG data directory convention."""
       data_dir = os.path.join(os.path.expanduser("~"), ".local", "share", "mdc-encyclopedia")
       return os.path.join(data_dir, "mdc_encyclopedia.db")
   ```
   Use `os.path.expanduser("~")` and `os.path.join` — no new dependencies. This follows the XDG Base Directory Specification on Linux (`~/.local/share/`). Do NOT add platformdirs or any new dependency.

2. Change line 39 from:
   ```python
   db_path = os.environ.get("MDC_ENCYCLOPEDIA_DB", "mdc_encyclopedia.db")
   ```
   to:
   ```python
   db_path = os.environ.get("MDC_ENCYCLOPEDIA_DB", get_default_db_path())
   ```

3. Update the "Created database" message to show the resolved absolute path so the user knows where it went:
