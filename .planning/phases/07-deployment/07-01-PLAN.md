<task type="auto">
  <name>Task 1: Create GitHub Actions workflow file</name>
  <files>.github/workflows/encyclopedia.yml</files>
  <action>
Create `.github/workflows/encyclopedia.yml` with the following structure:

**Name:** `Encyclopedia Pipeline`

**Triggers (3):**
1. `schedule`: cron `0 6 * * 1` (Monday 6:00 UTC / 1:00 AM EST)
2. `push`: branches `[master]`
3. `workflow_dispatch`: two boolean inputs:
   - `skip_enrichment` (default false) -- skip the enrich step entirely
   - `force_enrich` (default false) -- override the 25-dataset threshold

**Permissions:** `contents: write`, `pages: write`, `id-token: write`, `issues: write`

**Concurrency:** group `"pages"`, cancel-in-progress `false`

**Environment variables (workflow-level):**
- `MDC_ENCYCLOPEDIA_DB: ./mdc_encyclopedia.db` (keeps DB in repo root, not XDG default)
- `PYTHON_VERSION: '3.12'`

**Two jobs:**

**Job 1: `pipeline`** (runs-on: ubuntu-latest)
Steps:
1. `actions/checkout@v5`
2. `actions/setup-python@v5` with python-version from env and `cache: 'pip'`
3. `pip install .` -- install the package
4. **Pull latest data** -- `if: github.event_name != 'push'` -- run `mdc-encyclopedia pull`
5. **Run audit** -- `if: github.event_name != 'push'` -- run `mdc-encyclopedia audit`
6. **Detect changes** -- `if: github.event_name != 'push'` -- run `mdc-encyclopedia diff || true` (diff may fail on first run)
7. **Enrich new datasets (conditional)** -- `if: github.event_name != 'push'` -- shell script that:
   - Checks if `skip_enrichment` input is true -> skip
   - Checks if `ANTHROPIC_API_KEY` env var is empty -> echo message and exit 0
   - Counts new datasets by querying the DB directly with Python:
     ```
     NEW_COUNT=$(python3 -c "
     import sqlite3, os
     conn = sqlite3.connect(os.environ['MDC_ENCYCLOPEDIA_DB'])
     conn.row_factory = sqlite3.Row
     row = conn.execute('''
         SELECT COUNT(*) as cnt FROM changes
         WHERE change_type = 'added'
         AND detected_at = (SELECT MAX(detected_at) FROM changes)
     ''').fetchone()
     print(row['cnt'] if row else 0)
     conn.close()
     ")
     ```
   - If `NEW_COUNT >= 25` AND `force_enrich` is not true -> skip enrichment and create a GitHub issue via `gh issue create`:
     - Title: `"Manual enrichment needed: $NEW_COUNT new datasets"`
     - Body: Markdown with date, count, threshold explanation, and suggested manual commands (`mdc-encyclopedia enrich --model claude-sonnet-4-5-20250929` and `--model claude-haiku-4-5-20251001`)
     - Label: `enrichment`
   - Otherwise -> run `mdc-encyclopedia enrich --model claude-haiku-4-5-20251001`
   - Env vars for this step: `ANTHROPIC_API_KEY: ${{ secrets.ANTHROPIC_API_KEY }}`, `GH_TOKEN: ${{ secrets.GITHUB_TOKEN }}`
8. **Commit database updates** -- `if: github.event_name != 'push'` -- shell script that:
   - Configures git user as `github-actions[bot]`
   - `git add mdc_encyclopedia.db`
   - Checks if staged changes exist (`git diff --staged --quiet`), commits only if changes: `git commit -m "chore: update encyclopedia database [skip ci]"`
   - `git push`
9. **Export static site** -- always runs: `mdc-encyclopedia export -o site`
10. **Upload Pages artifact** -- `actions/upload-pages-artifact@v4` with `path: site/`

**Job 2: `deploy`** (runs-on: ubuntu-latest)
- `needs: pipeline`
- `environment`: name `github-pages`, url `${{ steps.deployment.outputs.page_url }}`
- Single step: `actions/deploy-pages@v4` with id `deployment`

IMPORTANT: The repo's default branch is `master`, NOT `main`. All trigger configurations must reference `master`.
IMPORTANT: Use `[skip ci]` in the DB commit message to prevent infinite push loops.
IMPORTANT: The `MDC_ENCYCLOPEDIA_DB` env var overrides the default XDG path so the DB is at `./mdc_encyclopedia.db` in the repo root.
