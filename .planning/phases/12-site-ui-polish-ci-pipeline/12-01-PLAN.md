   ```python
   from mdc_encyclopedia.registry import load_registry
   registry = load_registry()
   for ds in site_data["datasets"]:
       slug = ds.get("jurisdiction") or ""
       ds["jurisdiction_display_name"] = registry.get(slug, {}).get("display_name", slug.replace("-", " ").title()) if slug else ""
   ```

   - Unpack the 4th return value from `_extract_filter_options()`.
